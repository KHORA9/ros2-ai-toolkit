# ROS2 AI Toolkit

> Open-source tooling for building AI-powered robots with ROS2. 1,232x cost advantage over traditional ROS2 development.

[![License: MIT](https://img.shields.io/badge/License-MIT-purple.svg)](https://opensource.org/licenses/MIT)
[![ROS2](https://img.shields.io/badge/ROS2-Humble-blue.svg)](https://docs.ros.org/en/humble/)
[![Python 3.10+](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://www.python.org/downloads/)

Build AI-powered robots faster. This toolkit provides drop-in ROS2 nodes for vision, speech, navigation, and task planning — powered by modern AI models.

---

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/hanatra/ros2-ai-toolkit.git
cd ros2-ai-toolkit

# Install dependencies
pip install -r requirements.txt

# Launch a sample AI node
ros2 run ros2_ai_toolkit vision_node
```

---

## ✨ Features

- **Vision AI** — Object detection, pose estimation, visual SLAM integration
- **Speech** — Wake word detection, STT, TTS with LLMs
- **Navigation** — AI-enhanced path planning with dynamic obstacle avoidance
- **Task Planning** — LLM-powered task decomposition and execution
- **RAG for Robotics** — Connect your robot's manual and logs to a knowledge base
